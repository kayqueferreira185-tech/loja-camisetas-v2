# loja-camisetas-v2
Loja virtual de camisetas com HTML, CSS e JavaScript, incluindo dark mode e interações de UX.

# 🛍️ Lojas Kayque

Projeto de uma loja virtual de camisetas desenvolvido com foco em prática de **HTML, CSS e JavaScript**, evoluindo de uma versão simples para uma interface com **interatividade e foco em experiência do usuário (UX)**.

---

## 🚀 Sobre o projeto

Este projeto simula uma loja online, com estrutura semântica, design organizado e interações que melhoram a navegação do usuário.

Além da parte técnica, o foco foi aplicar conceitos que são utilizados em projetos reais de empresas, principalmente relacionados à **experiência do usuário (UX)**.

---

## ✨ Funcionalidades

* 🎨 Layout moderno e organizado
* 🌙 Dark mode (modo escuro)
* 🖱️ Efeitos de hover e feedback visual
* 🔗 Navegação entre páginas
* ⚡ Interatividade com JavaScript

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (DOM)

---

## 💡 Decisões de UX e impacto real

Durante o desenvolvimento, foram aplicadas algumas decisões que vão além do visual e impactam diretamente a experiência do usuário:

### 🌙 Dark Mode

Permite ao usuário escolher um tema mais confortável para leitura, especialmente em ambientes com pouca luz.

📊 **Impacto em empresas:**

* Reduz cansaço visual
* Aumenta tempo de permanência no site
* Melhora acessibilidade

---

### 🖱️ Feedback visual (hover e active)

Botões e elementos reagem ao passar o mouse ou clicar.

📊 **Impacto em empresas:**

* Deixa claro o que é clicável
* Reduz erros de navegação
* Aumenta taxa de interação

---

### 🧩 Estrutura limpa e organizada

Uso de HTML semântico e layout bem distribuído.

📊 **Impacto em empresas:**

* Facilita manutenção do código
* Melhora SEO (otimização para buscadores)
* Escala melhor em projetos maiores

---

### ⚡ Interatividade com JavaScript

Mudança de tema dinâmica sem recarregar a página.

📊 **Impacto em empresas:**

* Experiência mais fluida
* Sensação de aplicação moderna
* Redução de fricção na navegação

---

## 🎯 Problemas que este projeto resolve

Mesmo sendo um projeto simples, ele simula soluções para problemas reais como:

* ❌ Interfaces estáticas e pouco intuitivas
* ❌ Falta de feedback ao usuário
* ❌ Experiência de navegação pouco fluida
* ❌ Dificuldade de adaptação a preferências do usuário (tema claro/escuro)

👉 O projeto propõe uma interface mais interativa, clara e confortável.
